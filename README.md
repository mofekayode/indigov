# indigov

First run these commands to start the server

```bash
Terminal 1:

cd Server
# then
nmp install
# then
npm start

Terminal 2:

cd Client
nmp install
# then
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
